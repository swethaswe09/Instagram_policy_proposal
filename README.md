# Instagram_policy_proposal
Identified four real enforcement gaps in Instagram's moderation system through direct observation. Developed a structured policy proposal covering mandatory age labeling, non-consensual image prohibition, bio link accountability, and an Aadhaar-linked harassment accountability framework for India.
